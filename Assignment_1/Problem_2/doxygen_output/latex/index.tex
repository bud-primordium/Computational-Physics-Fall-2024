\chapter{24-\/Game Solver}
\hypertarget{index}{}\label{index}\index{24-\/Game Solver@{24-\/Game Solver}}
An enhanced version of the 24-\/game solver using recursive search and pruning.

This program allows users to solve the 24-\/game using the following features\+:
\begin{DoxyItemize}
\item Recursive search with pruning
\item Progress bar to monitor search progress
\item Open\+MP parallelization for multi-\/core systems
\item Optimization for commutative operations (addition and multiplication)
\end{DoxyItemize}

Users can either use default settings or customize them according to their needs. The program outputs include\+:
\begin{DoxyItemize}
\item The first valid solution for 24
\item Detailed recursive steps and expressions
\end{DoxyItemize}
